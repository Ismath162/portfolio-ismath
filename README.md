### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Ismath162/portfolio-ismath-master.git
   cd portfolio-ismath-master
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8080
   
   # Using Node.js
   npx serve
   ```
